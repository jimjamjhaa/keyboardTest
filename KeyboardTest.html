Keyboard Test Compatibility Qwerks

  This keyboard Test is written in HTML5 and as a result is incompatible with Internet Explorer versions 1-8, Firfox versions 1-25 and Chrome versions 1-25

  If number lock is active the number pad will return an alternate value key press.

  Some keys have alternate primary functions within the browser that are outside of the scope of this page.

  • The windows key will open the start menu, a second press will return focus to the page.
  • The Fn key has no functionality within the browser so it does not appear in on this page
